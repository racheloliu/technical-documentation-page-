Introduction

What you should already know

This reference page covers the basics of web literacy, so you do not need to know any code.

What you should be familiar with:

  • How to use a web browser to access the internet
  • What a website is and how to access them via an address bar or search engine
  • That HTML, CSS, and JavaScript are coding markups and languages, even if you don't know how they work
HTML and CSS

HTML is a markup language, as evidenced by its name, Hyper Text Markup Language. A markup language is what is used to write out text for a webpage and have it show up in your browser.

Depending on the syntax used, HTML can categorize text in a hierarchy of header, paragraph, list, etc., and organize textual information

HTML can be linked to CSS to adjust its appearance, which is why HTML and CSS are often spoken of in tandem.

!DOCTYPE html

CSS stands for Cascading Style Sheets, and through links, can change everything from the background, images, and even text itself through, for example, color and font.

body { background-color: blue; }

Often if a developer is coding HTML, they are referencing and ajusting CSS code at the same time, and vice versa.

#main-doc {} .main-section {}

JavaScript

JavaScript is a programming language.

For websites, it can communicate with HTML/CSS elements for interactivity.

getElementById()

JavaScript can reference files outside of what you see on a website, and can even change HTML or CSS.
